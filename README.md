## Database Organization

_There are two tables named SURVEYS and QUESTIONS._

Survey fields:
* SURVEY_ID
* USERNAME
* EMAIL_ADDRESS
* TOPIC

Question fields:
* QUESTION_ID
* DESCRIPTION
* TYPE
* ANSWER
* FK_SURVEY_QUESTION
